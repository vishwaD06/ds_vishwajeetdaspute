**Project Overview**
This analysis explores how market sentiment (Bitcoin Fear & Greed Index) affects trader performance using historical trading and sentiment data. Data is cleaned, merged, and visualized to identify key patterns in profit and activity across sentiment cycles. Results offer actionable insights for trading strategy and risk management.
